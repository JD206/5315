What is Docker

Docker runs containers which bundle their own tools, libraries, and configs. source: www.docker.com

What is Nginx

Nginx is a web server that was released in 2004, with a supporting company founded in 2011. source: www.nginx.com

What is VM (virtual machine)

Virtual Machines are software defined computers that allow for flexible, SW defined allocation of the equivalent of physical hardware. source: VMware.com

Who is Cloud Computing Service Provider - Digitalocean

Digital Ocean is an IaaS company based in NYC. Provides VMs (droplets) and storage. source: www.digitalocean.com
